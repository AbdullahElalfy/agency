# Agency Website 🌐

A modern and responsive **Agency landing page** built with pure **HTML, CSS, and JavaScript**.  
This project was created as a training project to practice frontend development skills without using any frameworks or libraries.

---

## 🚀 Features
- Fully responsive layout for all devices 📱💻
- Clean and modern UI design ✨
- Smooth animations and hover effects 🎨
- Fast loading with pure HTML, CSS, and JS ⚡

---

## 🛠️ Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
